# Python basics for engineering
This is my first repository for learning and practicing Python fundamentals with engineering examples.

## What this repository contains 
This repository includes simple Python programs that solves basics engineering style problems such as:
 - Swaping two numbers
 - Temperature conversion (Celsius to Fahrenheit)
 - Area of a circle calculation
 - Simple interest calculation
 - Splitting a decimal number into integer and fractional parts

## Topic covered 

 - Taking input from the user
 - Type conversion (string, int, float)
 - Arithmatic operations
 - Using mathematical formulas in Python
 - Basic problem-solving logic

## How to run the programs 

1. Make sure Python is installed on your system.
2. Downloac or clone this repository.
3. Open a terminal or command prompt in the project folder.
4. Run any file using:

   pthon filename.py

   Example:
   python temperature_converter.py
